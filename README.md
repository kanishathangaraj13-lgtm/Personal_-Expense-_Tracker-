# Personal Expense Tracker

## Project Description

The Personal Expense Tracker is a web-based application that helps users manage their daily income and expenses. It allows users to record transactions, categorize expenses, monitor spending, and view financial summaries for better budgeting and money management.

## Features

- User Registration
- User Login
- Add Income
- Add Expenses
- Expense Categories
- Transaction History
- Monthly Expense Summary
- Budget Tracking
- Database Management

## Technologies Used

- HTML

- JavaScript
- MySQL
