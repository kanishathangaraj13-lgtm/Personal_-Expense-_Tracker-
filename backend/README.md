# Backend

## Description

The backend of the Personal Expense Tracker manages the application's business logic and database operations. It processes user requests, stores income and expense records, and retrieves financial data from the MySQL database.

## Responsibilities

- User Registration
- User Login Authentication
- Income Management
- Expense Management
- Transaction History
- Report Generation
- Database Management

## Technologies Used

- JavaScript (Node.js)
- MySQL

## Database Tables

- Users
- Income
- Expenses

## Future Enhancements

- Secure Authentication
- Budget Notifications
- Data Export (PDF/Excel)
- Dashboard Analytics
